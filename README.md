# Étude de l’impact du coefficient de coascendance en forensique

Dans le cadre de la science forensique projet de Master de mathématiques appliqués s'intéresse au rapport de vraissemblance en fonction du coefficient de coascendance. Cet outil permet de comparer le poids d’une preuve en faveur d’une hypothèse par rapport à une autre (généralement l'accusation face à la défense). Dans notre cas, la preuve correspond à la trace génétique (retrouvée sur la scène de crime) et on compare l'hypothèse que le coupable et le suspect ont les même profils génétiques contre l'hyptohèse qu'ils n'ont pas le même.

## Installation

## Lancement du programme

Ouvrir un terminal là où sont stockés les fichiers python, et effectuer dans un terminal la commande suivante:
 
```bash

```
